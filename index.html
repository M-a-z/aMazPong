MazPong - pong game playable over network


General info

I started this project ages ago (2006?) in order to try learning some graphichs generation using C and glut. Unfortunately my schedules have been really on and off, so this project has been abandoned, restarted, abandoned, forgotten, revives and so on... Now it is 2013, and I am wondering if I should try to make something working out of this... If you have nothing else to do, feel free to fork the sources and see what you can do with them. I am gladly getting suggestions/fixes etc. =)

How to obtain it

Development version of MazPong is found from github.
It can also be fetched using git client by issuing command

git clone https://github.com/M-a-z/aMazPong.git
